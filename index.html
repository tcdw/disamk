AddMusicK 1.0.X SPC Disassembler

Danger alert! You should check in on some of those fields below.

You can convert your AddMusicK 1.0.X SPC back into MML and BRR samples.

Since Oct 6, 2023, MIDI file is also exported. Be aware this feature is still experimental, and most hex-command effects are not implemented yet.

Select your SPC file and press Start, then the disassembled result will be downloaded automatically from your browser.

Only unmodified versions of AddMusicK 1.0.0 ~ 1.0.9 SPC is guaranteed to be supported. For AddMusic 4.05/M/etc SPC, use NintSPC; you can also use my own build of NintSPC for accurate qXY command conversions and additional AddMusic commands support (source).

Important Notice

This tool is purely frontend-based, which won't upload any data into our server. You can get source code here.
